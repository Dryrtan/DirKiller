# DirKiller
Um scanner web para detectar vulnerabilidade Directory Traversal, feito para fins de conhecimento e tempo livre.

# Como usar?
```
git clone https://github.com/Zian25/DirKiller
cd DirKiller
python3 dirkiller.py -h
```
